# Instructions, after downloading the files into a local directory:

0. install nodejs (LTS) https://nodejs.org/en/download
1. run `npx shopify app deploy`
2. run `npm install`
3. run `npm start`
4. Click install link in partner apps / new dev dashboard and follow the flow,


The cli will print the session containing the accessToken which is what you use, ensure it has the offline property
